📋 PASOS PARA DESPLEGAR EN VERCEL

1️⃣  Abre: https://vercel.com

2️⃣  Click en "Sign up with GitHub"
   - Autoriza Vercel para acceder a tu GitHub
   - Completa tu perfil

3️⃣  En el dashboard, click en "Add New"

4️⃣  Selecciona "Project"

5️⃣  En "Import Git Repository":
   - Busca: app-compraventa
   - Haz click en ella
   - Click "Import"

6️⃣  Configura el Proyecto:
   
   Vercel sugiere automáticamente:
   Project Name:        app-compraventa (está bien)
   Framework Preset:    Vite ✅ (asegúrate que sea Vite)
   Root Directory:      frontend ✅ (MUY IMPORTANTE)
   Build Command:       npm run build (está bien)
   Output Directory:    dist (está bien)
   Install Command:     npm install (está bien)

   Si no aparecen, ve a "Project Settings" y configúralos

7️⃣  Environment Variables:
   
   Scroll hacia abajo a "Environment Variables"
   
   Agrega esta variable:
   
   - Name: VITE_API_URL
   - Value: https://compraventa-backend.onrender.com/api
   
   (Reemplaza con tu URL real de Render)
   
   ⚠️  IMPORTANTE: Incluye "/api" al final

8️⃣  Click en el botón azul "Deploy"

9️⃣  ESPERA 5-10 MINUTOS mientras Vercel compila

   Verás:
   - "Building..."
   - Varios logs
   
   Cuando termine verás:
   ✓ "Congratulations! Your project has been successfully deployed"
   ✓ Una URL como: https://compraventa-vehiculos.vercel.app
